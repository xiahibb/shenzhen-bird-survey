## ✨ Core Features

| Feature | Description |
|---------|-------------|
| 🕒 Global Countdown | Supports pause/resume/restart with a pop-up reminder when time's up. |
| 🐦 Pinyin Bird Search | Quickly find bird species by typing the first letters of their Pinyin (e.g., `zhgz` → 中华鹧鸪). |
| 📊 Data Overview | One-click summary of total points, species, individuals, and habitat types. |
| 🐣 Breeding Stats | Automatically categorizes breeding evidence: Confirmed/Probable/Flyover & Heard. |
| 📎 One-Click Export | Generates a CSV file for easy reporting. |
| 💾 Local Storage | All data is saved on your device; refresh or close the page without losing anything. |

## 🎯 Who is this for?

- CBBS volunteers in Shenzhen for quick field recording.
- Birdwatching enthusiasts for personal logs.
- Bird clubs, training sessions, and student field practices.
