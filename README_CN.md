# shenzhen-bird-survey
🐦 深圳繁殖鸟类调查 · 野外速记工具 · 倒计时 · 拼音搜鸟 · 一键导出

![Netlify](https://img.shields.io/badge/hosted%20on-Netlify-00c7b7)

### ✨ 核心功能

| 功能 | 说明 |
|------|------|
| 🕒 全局倒计时 | 支持暂停/继续/重新开始，归零弹窗提醒 |
| 🐦 拼音搜鸟 | 输入首字母快速匹配鸟种（如 `zhgz` → 中华鹧鸪） |
| 📊 数据概览 | 总样点/鸟种/数量/生境数，一键查看详情 |
| 🐣 繁殖统计 | 确定繁殖/可能繁殖/飞过听见，标签区分 |
| 📎 一键导出 | 生成 CSV 表格，方便上报 |
| 💾 本地存储 | 数据仅存手机，刷新不丢失 |

### 🎯 适合谁用

- 深圳 CBBS 调查志愿者野外速记
- 观鸟爱好者个人记录
- 鸟会培训、社团实践
